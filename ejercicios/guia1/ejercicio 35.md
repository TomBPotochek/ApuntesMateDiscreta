# b
probar $xy \leq x \leq x+y$
recordanos que (*definicion de orden*)
$$
A \leq B \iff AB = A \ \wedge \ A+B = B
$$
demostracion:
$$
\begin{split}
xy \leq x \leq x+y &= \\
(xy \leq x) \ \wedge (x \leq x+y) &= \\
xyx = xxy = xy \ \wedge \ x(x+y) = x \\
x(x+y)=xx +xy = x\underbrace{(1+y)}_{=1} = x
\end{split}
$$
